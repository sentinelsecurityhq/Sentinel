# Security Policy

## Reporting a Vulnerability

Sentinel Security takes security seriously. If you discover a vulnerability in our software, we appreciate your help in disclosing it responsibly.

Please report vulnerabilities by emailing **oxajamal@gmail.com** with the subject line "Security Vulnerability Report."

We will acknowledge receipt within 48 hours and provide an initial assessment within 5 business days.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x (development) | Yes |

## Scope

This policy applies to all code and infrastructure maintained under the Sentinel Security GitHub organization.
